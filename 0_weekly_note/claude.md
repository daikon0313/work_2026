# 0_weekly_note - 週次ノート

週次の振り返りとノートを管理するディレクトリです。

## プロジェクト概要

週次で振り返りを行い、学びや気づきを記録するためのシステムです。テンプレートとスクリプトを使用して、効率的にノートを作成できます。

## ディレクトリ構成

```
0_weekly_note/
├── README.md          # プロジェクトの説明
├── claude.md          # Claude用の指示
├── template.md        # 週次ノートのテンプレート
├── shuuhou.sh         # ノート作成スクリプト
└── notes/             # 週次ノートの保存場所
```

## 使い方

### 新しい週次ノートの作成

```bash
cd 0_weekly_note
./shuuhou.sh
```

スクリプトを実行すると、以下の処理が行われます：
1. 現在の週番号に基づいてファイル名を生成
2. テンプレートをコピー
3. 日付情報を自動入力
4. `notes/` ディレクトリに保存

### テンプレートの構成

週次ノートには以下のセクションが含まれます：

- **今週の目標**
- **達成したこと**
- **学んだこと**
- **課題・改善点**
- **来週の目標**

## ベストプラクティス

### ノート作成のタイミング

- 週の初め（月曜日）：今週の目標を設定
- 週の終わり（金曜日）：振り返りを記録

### 記録のコツ

1. **具体的に書く**
   - ❌ 「勉強した」
   - ✅ 「Snowflakeの公式ドキュメントを3章まで読んだ」

2. **数値を含める**
   - 達成率、時間、件数など具体的な数値を記録

3. **感情も記録**
   - 良かったこと、難しかったこと、楽しかったこと

4. **次週へのアクション**
   - 来週の目標は具体的かつ実行可能なものにする

## カスタマイズ

### テンプレートの変更

`template.md` を編集して、自分に合った形式にカスタマイズできます。

### スクリプトの修正

`shuuhou.sh` を編集して、ファイル名の形式や保存場所を変更できます。

## トラブルシューティング

### スクリプトが実行できない

実行権限を付与してください：

```bash
chmod +x shuuhou.sh
```

### テンプレートが見つからない

`template.md` が同じディレクトリに存在することを確認してください。
