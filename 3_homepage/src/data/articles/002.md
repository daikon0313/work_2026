# dbtで実現するデータ変換の自動化

## dbtとは
dbt (data build tool) は、SQLベースのデータ変換ツールです。

## 主な機能
- **モジュール化されたSQL**: 再利用可能なSQLモデルの作成
- **テスト機能**: データ品質の自動チェック
- **ドキュメント生成**: 自動的にデータカタログを作成

## サンプルコード
```sql
-- models/staging/stg_orders.sql
with source as (
    select * from {{ source('raw', 'orders') }}
)

select
    order_id,
    customer_id,
    order_date,
    total_amount
from source
```

## まとめ
dbtを活用することで、保守性の高いデータパイプラインを構築できます。
